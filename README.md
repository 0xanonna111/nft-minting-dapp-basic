# NFT Minting DApp Basic

This project helps beginners learn how NFTs work and how to mint one using a Web3 app.

You will learn:
- What an NFT is
- How to connect a wallet
- How to call a smart contract
- How to mint a token
- How to read token data

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/nft-minting-dapp-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- nft.js = NFT contract functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Mint NFT
- Show token ID
- Show owner address
- Simple smart contract use

## Learning Goal
This project helps new developers understand NFTs and smart contract interaction.
